# Project Sovereign

Static marketing site for Project Sovereign, a European self-hosted document
suite initiative.

The public site is generated by a small custom Node build. Source content lives
in `src/`, the renderer lives in `scripts/build.js`, and production output is
written to `dist/`.

## Local Preview

```bash
node scripts/build.js
python3 -m http.server 4173 --bind 127.0.0.1 --directory dist
```

Open `http://127.0.0.1:4173/`.

## Publish

Vercel runs `npm run build` and serves `dist/`. The contact form posts to
`/api/contact`, which forwards validated requests to the NatorOS webhook stored
in `NATOROS_CONTACT_WEBHOOK_URL`.
